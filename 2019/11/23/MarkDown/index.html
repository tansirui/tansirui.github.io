需求环境

Nodejs 安装长期支持版本即可

安装

npm类似于apt-get

npm install -g hexo-cli

hexo init “floder” 初始化站点的目录

npm install 切换到站点目录安装依赖

hexo s 启动本地站点

_config.yml 该文件下修改主页相关配置

hexo clean 清除生成的数据

hexo g 生成新的HTML界面

博客中的文章存在source_posts文件夹下，文件开头为配置信息

部署到github需要注意，仓库名必须和用户名相同

npm install hexo-deployer-git –save 安装插件

hexo d 推送文件到github
